The following links points to useful resources used in the project.

-   [Nuxt](https://nuxt.com/)  
    The framework used, based on [Vue.js](https://vuejs.org/guide/introduction.html)
-   [Nuxt Icons](https://icones.js.org/)  
    List of icons that can be used inside the project.  
    Example: `<icon name="solar:login-3-bold"></icon>`. Any icon in the website can be used right away.
-   [VueUse](https://vueuse.org/guide/)  
    A Vue plugin that has useful utilities (I/O, filesystem, responsive screens, animation, network...)
-   [Pinia store](https://pinia.vuejs.org/introduction.html)  
    State management package used, perfectly integrated with Nuxt framework.
-   [Postgres npm](https://www.npmjs.com/package/postgres)  
    Package used to connect to production PostgreSQL based databases.
-   [Apexcharts](https://apexcharts.com/docs/installation/)  
    Charting library used in the project.
-   [Mongodb npm](https://www.npmjs.com/package/mongodb/)  
    Package used to connect to analytics NoSQL database (CosmosDB).
